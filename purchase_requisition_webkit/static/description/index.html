Purchase Requisition Report using Webkit Library

This module was written to replace the legacy rml Requisition report by a webkit report, and allow you to benefit from a more robust technology.

Installation

To install this module, you need to:

  • Download additionally the module 'base_headers_webkit' available in OCA's 'webkit-tools' repository (https://github.com/OCA/webkit-tools/tree/7.0)

Configuration

This module does not require any additional configuration.

Usage

This module replaces the Requisition form.
